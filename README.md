# PDF画像化 & 圧縮ツール - ローカル実行ガイド

このツールを自分のPC（Windows）で動かすための手順です。

## 1. 準備するもの
- **Python**: まだインストールしていない場合は、[python.org](https://www.python.org/) から最新版をインストールしてください。
- **このフォルダのファイル一式**: `app.py` や `poppler` フォルダが揃っていることを確認してください。

## 2. セットアップ手順

1.  **コマンドプロンプト（またはPowerShell）を開く**
    - `Windowsキー + R` を押し、`cmd` と入力してエンターを押します。

2.  **このフォルダに移動する**
    - コマンドプロンプトで以下のコマンドを入力して移動します（フォルダのパスは環境に合わせて調整してください）：
      ```cmd
      cd /d "c:\Users\marak\OneDrive\デスクトップ\010_まちのAI屋さん\070_サービス作成したい★\073_Webアプリを作成したい\18_PDF_tool"
      ```

3.  **必要なライブラリをインストールする**
    - 以下のコマンドをコピー＆ペーストして実行してください：
      ```cmd
      pip install -r requirements.txt
      ```

## 3. アプリの起動

セットアップが完了したら、以下のコマンドでアプリを起動します：

```cmd
streamlit run app.py
```

実行後、ブラウザが自動的に開き、ツールが使えるようになります。

---
**注意**: Poppler（PDF変換エンジン）はこのフォルダ内の `poppler` ディレクトリを使用するように設定されているため、別途インストールする必要はありません。
